# Repair Shop — Work Order flow mockups

Static SVG mockups for the work-order lifecycle (create → build in-house + sublet → authorize → execute → parts/PO → invoice → pay → post). On-brand with the Squibbon client (Spike / Flowbite-React, purple `#c479c2` / teal `#16CDC7`, Plus Jakarta Sans). Built against the field spec — open each `.svg` in a browser; export PNG/PDF for Notion.
